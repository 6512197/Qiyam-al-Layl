THIS IS A RAMADAN PROJECT :
 A small webpage that computes a precise "third" point of the night for any location and date (works in browser or Node).  
information on the calculations made based on latitudes : 
Fajr begins at true dawn (al-fajr al-ṣādiq) — when horizontal light first appears on the eastern horizon.
Astronomically, this corresponds to the sun being a certain number of degrees below the horizon. That’s what the “angle” (e.g., -18°) refers to.
Common Angles Used for Fajr
Angle	What It Means	Effect
-18°	Sun 18° below horizon	Earlier Fajr
-17°	Slightly less strict	Slightly later
-15°	More conservative in bright regions	Noticeably later
-12°	Used in high-latitude adjustments	Much later
The larger the negative number, the earlier Fajr begins.
Why the Difference?
Because:
Atmospheric conditions differ
Light pollution affects visibility
Classical scholars described dawn visually, not astronomically
Modern measurements show dawn brightness varies by location
Some contemporary researchers argue -18° may be too early in many urban environments and suggest -15° or -16° may better match visible dawn.
There is no universal global agreement.
